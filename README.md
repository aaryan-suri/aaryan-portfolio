# aaryan-portfolio

Personal portfolio site for **Aaryan Suri**. Built with Next.js (App Router) and Tailwind CSS, with project pages under `/projects/[slug]`.

## Tech

- Next.js (App Router), React, TypeScript
- Tailwind CSS v4
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy / environment

Optional: set `NEXT_PUBLIC_SITE_URL` (no trailing slash) so OG metadata and the sitemap use the correct canonical domain.
